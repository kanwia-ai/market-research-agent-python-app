# Intake Form Fields

## Required Fields (must have)

| Field | Label | Type | Description |
|-------|-------|------|-------------|
| offering_what | What are you offering? | text | The product/service being researched |
| offering_problem | What problem does it solve? | textarea | The pain point it addresses |
| target_customer | Who is your target customer? | text | Description of ideal customer |
| geography | Target geography | text | Countries/regions to focus on |
| primary_question | #1 question to answer | textarea | The main thing they want to learn |
| depth | Research depth | dropdown | "overview", "thorough", "deep_dive" |

## Optional Fields (nice to have)

| Field | Label | Type | Description |
|-------|-------|------|-------------|
| offering_delivery | How will it be delivered? | text | Online, in-person, hybrid, etc. |
| offering_pricing_model | Pricing model | text | One-time, subscription, etc. |
| known_competitors | Known competitors | text | Comma-separated list |
| customer_conversations | Have you talked to customers? | textarea | What they've learned so far |
| opportunity_thesis | Why now? | textarea | Why this opportunity exists |
| stage | Business stage | dropdown | "exploring", "committed", "expanding" |
| kill_criteria | What would kill this idea? | textarea | Deal-breakers to look for |

## Depth Levels

- **overview**: Quick sweep, find the obvious. 5-10 min research per agent.
- **thorough**: Comprehensive. Follow leads, check secondary sources. 15-20 min per agent.
- **deep_dive**: Exhaustive. Leave no stone unturned. 30+ min per agent.
